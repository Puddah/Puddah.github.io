About me

Hi, my name is Tim, I'm a software developer living in Emden, which is located at the beautiful north sea coast of germany.

I have written code in Java, C, C++, C#, Javascript and Ruby. I have started to code in Pascal and Basic in the mid 1990s and lately played around a bit in Elixir/Erlang, Haskell and elm. I'm also into test driven development, domain driven design and simple design.

When I'm not writing code I do a lot of sports either on the basketball court taking shots or in the gym lifting weights. I love traveling with my beautiful girlfriend Steffi and sometimes I just have a beer (maybe some more) with my friends.

Projects

These are some Projects I have worked on which are reachable on the internet:

portlogs.com
frontend and backend engineer, operations.
Portlogs is a networking website for the maritime business. Shipping companies, agencies, shipowner, etc. are able to maintain their relationships worldwide easily and transparently.
It is written in ruby and runs on rails behind nginx. The overall architecture follows cqrs principles with event-sourcing. The events and readmodels life in mongodb.
rendezvous
open source side project to lern elixir
Implementation of the Rendezvous or Highest Random Weight (HRW) hashing algorithm in the Elixir Programming Language (elixir-lang.org).
Besides elixir I also tried out travis-ci and coveralls.
FitEx
open source side project to lern elixir macros
FitEx provides a simplified function definition syntax in elixir. I basically created it to try out elixirs meta programming capabilities.
rcqrs
open source project extracted from portlogs
Rcqrs is a framework which implements the cqrs pattern in ruby. (see cqrsinfo.com)
